# **App Name**: PlanPal

## Core Features:

- User Profiles: Allow users to create profiles with allergies, dietary restrictions, preferences, and calendar availability.
- Friend Management: Enable users to send, accept, and manage friend requests.  Set nicknames for friends.
- Plan Creation and Management: Enable users to create plans by defining: name, description, event time, location, city, and price range. The plan overall status needs to be tracked.
- Participant Management: Allow hosts to invite friends to plans and track their confirmation status (pending, confirmed, declined).
- AI-Powered Event Suggestions: Use an AI tool to generate tailored event propositions based on the host's plan details (city, time) and participants' specificities (allergies, preferences, availability) when generating results for the location/activity suggestions.
- Split Payment Management: Manage split payments, connect bank cards, split the predicted budget, and take preauthorization from participants.

## Style Guidelines:

- Primary color: Calm blue (#42a5f5) to convey trust and reliability.
- Secondary color: Light gray (#f5f5f5) for backgrounds and subtle contrasts.
- Accent: Teal (#009688) for a fresh and modern feel for buttons.
- Clean and readable font for the entire application, focusing on clarity.
- Simple, intuitive icons to represent different event types and preferences.
- A clear and intuitive layout.
