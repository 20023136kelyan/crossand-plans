## Usage Guidelines

### Report Categories

1. Growth Metrics
   - New user registration
   - User activation rate
   - Geographic distribution
   - Acquisition channels

2. Engagement Analysis
   - Daily/Monthly active users
   - Session duration
   - Feature usage
   - Content creation

3. Retention Reports
   - Cohort analysis
   - Churn prediction
   - Reactivation rates
   - Engagement patterns

4. Conversion Tracking
   - Free to premium
   - User to creator
   - Purchase patterns
   - Upgrade triggers
