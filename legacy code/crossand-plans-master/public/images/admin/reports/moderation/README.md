## Usage Guidelines

### Report Types
1. Daily Moderation Summary
   - Content review stats
   - Action breakdown
   - Response times
   - Moderator performance

2. Content Violation Reports
   - Violation categories
   - Evidence documentation
   - Action taken
   - Appeal status

3. User Reports Analysis
   - Report patterns
   - User behavior
   - False positive rates
   - Resolution stats

4. Audit Logs
   - Moderator actions
   - Decision timeline
   - Evidence chain
   - Resolution path

### Implementation Guidelines
- Include timestamp on all reports
- Maintain clear evidence chain
- Support multi-language
- Enable quick filtering
- Include moderator notes
- Track appeal status
