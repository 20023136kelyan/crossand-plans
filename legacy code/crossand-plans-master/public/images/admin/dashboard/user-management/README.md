## Usage Guidelines

- Use consistent color coding for roles
- Maintain recognizable status indicators
- Keep verification badges prominent but not overwhelming
- Ensure default images are professional and neutral
- Support both light and dark themes
- Use vector formats for all icons
- Keep file sizes optimized
