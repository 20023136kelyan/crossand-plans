## Usage Guidelines

- Use consistent color coding:
  - Green: Approved/Safe
  - Yellow: Pending/Warning
  - Red: Rejected/Dangerous
- Include both light and dark variants
- Maintain clear visual hierarchy
- Keep placeholder images neutral and professional
