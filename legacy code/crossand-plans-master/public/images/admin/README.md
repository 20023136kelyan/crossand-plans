# Admin Directory

This directory contains images specific to admin functionality.

## Directory Structure

```
admin/
├── dashboard/         # Dashboard UI elements
│   ├── icons/        # Dashboard icons
│   ├── charts/       # Chart templates
│   └── backgrounds/  # Dashboard backgrounds
└── reports/          # Report assets
    ├── templates/    # Report templates
    ├── headers/      # Report headers
    └── exports/      # Export templates
```

## Image Requirements

### Dashboard Images
- Icons: 24x24px, 36x36px (2 sizes)
- Charts: SVG format preferred
- Backgrounds: 1920x1080px max

### Report Images
- Headers: 800x200px
- Logos: Vector formats
- Templates: PDF and PNG formats

## Admin Features Support

### 1. Analytics Dashboard
- Chart templates
- Status indicators
- Trend arrows
- Activity icons

### 2. Content Moderation
- Placeholder images
- Warning icons
- Status badges
- Review markers

### 3. User Management
- Role icons
- Status indicators
- Verification badges
- Warning symbols

### 4. Report Generation
- Report headers
- Export templates
- Watermarks
- Page backgrounds

### 5. System Status
- Status indicators
- Alert icons
- Progress indicators
- Loading animations

## Naming Conventions

```
admin-[feature]-[type]-[dimension].[extension]

Examples:
admin-dashboard-icon-24x24.svg
admin-report-header-800x200.webp
admin-status-alert-36x36.svg
```

## Guidelines

1. Use consistent styling
2. Maintain dark/light mode versions
3. Follow accessibility guidelines
4. Keep file sizes minimal
5. Use vector formats when possible

## Security Notes

- Keep sensitive graphics separate
- Use proper permissions
- Implement access control
- Regular security audits
- Monitor usage patterns
