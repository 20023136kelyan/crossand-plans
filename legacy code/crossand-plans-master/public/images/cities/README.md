# Cities Directory

This directory contains images for city covers and landmarks.

## Required Images

1. Cover Images (1920x1080px):
- city-default-cover-1920x1080.webp
- city-[cityname]-cover-1920x1080.webp

2. Landmarks (1280x720px):
- city-[cityname]-landmark-[name]-1280x720.webp

3. Thumbnails (500x500px):
- city-[cityname]-thumb-500x500.webp

## Image Guidelines

- Use recognizable skylines or landmarks
- Include both day and night variations when possible
- Ensure good contrast for text overlay
- Maintain consistent style across cities
- Use high-quality aerial or architectural photography

## Naming Convention

Replace [cityname] with lowercase, hyphenated city names:
- new-york
- san-francisco
- los-angeles
- etc.

Example:
```
city-new-york-cover-1920x1080.webp
city-new-york-landmark-central-park-1280x720.webp
city-new-york-thumb-500x500.webp
```
