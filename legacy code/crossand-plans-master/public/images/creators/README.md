# Creators Directory

This directory contains images for creator profiles and content.

## Directory Structure

```
creators/
├── profiles/           # Profile pictures
├── covers/            # Cover images
└── content/           # Creator-generated content
```

## Required Images

1. Profile Pictures (500x500px):
- creator-[username]-profile-500x500.webp
- creator-default-profile-500x500.webp

2. Cover Images (1920x1080px):
- creator-[username]-cover-1920x1080.webp
- creator-default-cover-1920x1080.webp

3. Content Images:
- Various sizes based on usage
- Follow general naming convention

## Image Guidelines

### Profile Pictures
- Professional headshots preferred
- Clear face visibility
- Neutral background
- Square aspect ratio

### Cover Images
- Lifestyle or action shots
- Represent creator's specialty
- Good contrast for text overlay
- 16:9 aspect ratio

### Content Images
- High quality, original content
- Properly credited if sourced
- Optimized for web delivery

## Naming Convention

Replace [username] with creator's unique username:
```
creator-johndoe-profile-500x500.webp
creator-johndoe-cover-1920x1080.webp
creator-johndoe-content-[description]-[dimension].webp
```

## Verification Badges

- Verified creators get priority placement
- Include badge overlay in profile display
- Store badge graphics in /logos directory
